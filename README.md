# Actividad 1 (Velocidades Lineales y angulares) Robot Planar 3GDL

Este repositorio contiene el desarrollo de la obtención del vector de velocidades lineal y angular de un robot planar de 3 grados de libertad (3 GDL) con articulaciones rotacionales.

Este repositorio contiene:
- Código MATLAB para el cálculo del Jacobiano
- Obtención de velocidades lineales y angulares
- Reporte explicativo del procedimiento
